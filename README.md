# PTG ProTraderGain

PTG ProTraderGain is an advanced trading risk management platform designed to help traders monitor and manage their risk exposure across multiple exchanges.

## Features

- **Risk Management**: Monitor and control your trading risk in real-time
- **Exchange Connections**: Connect to multiple cryptocurrency exchanges
- **Position Tracking**: View all your open positions in one place
- **Trading Insights**: Get personalized insights about your trading patterns
- **Multi-language Support**: Available in English and Portuguese

## Technology Stack

- React
- TypeScript
- Tailwind CSS
- Supabase
- Recharts
- i18next for internationalization

## Getting Started

1. Clone the repository
2. Install dependencies with `npm install`
3. Start the development server with `npm run dev`

## Deployment

This project is deployed on Netlify.
